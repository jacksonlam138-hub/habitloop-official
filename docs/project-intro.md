> 用游戏化的方式，让打卡不再枯燥，找到志同道合的伙伴一起成长

## 产品简介

HabitLoop 是一款游戏化的习惯养成应用。我们相信，最好的动力来自**有趣的体验**和**伙伴的支持**。

通过积分、成就、徽章和社群系统，让养成习惯变得有趣，而不是负担。

## 三大预设活动

### 🏋️ 健身/运动打卡
- **目标类型**：跑步、瑜伽、健身房、每日步数等
- **自动追踪**：支持Apple Health同步
- **社群活动**：月跑挑战、减脂营、健身小队
- **特色功能**：运动数据可视化、打卡晒图

### 🤖 AI学习
- **目标类型**：学习AI工具、AI绘图、Prompt Engineering
- **学习路径**：从入门到精通的渐进式课程
- **每日任务**：每日练习、实战项目打卡
- **社群活动**：AI挑战赛、作品展示、经验分享

### 🌍 英语学习
- **目标类型**：背单词、听力练习、口语打卡
- **游戏化设计**：闯关模式、连续打卡奖励
- **每日任务**：单词背诵、听力训练、口语练习
- **社群活动**：英语角、配音比赛、阅读小组

## 核心功能

### 1️⃣ 智能目标设定
输入你的大目标，HabitLoop 自动帮你拆解成每日可执行的任务
```
目标：3个月减重5公斤
↓ 自动拆解
每日任务：运动30分钟 + 记录饮食 + 睡前打卡
```

### 2️⃣ 连续打卡奖励
| 连续天数 | 奖励 |
|---------|------|
| 3天 | 银色徽章 |
| 7天 | 金色徽章 + 双倍积分 |
| 30天 | 钻石徽章 + 解锁专属皮肤 |
| 100天 | 传奇徽章 + 终身VIP特权 |

### 3️⃣ 社群活动
任何用户都可以发起打卡活动：
- "21天早起挑战"
- "30天阅读马拉松"
- "每日健身小队"
- "AI工具学习营"

找到你的同频伙伴，一起坚持！

### 4️⃣ 排行榜与PK
- 全球排行榜：看看谁是最强打卡王
- 伙伴PK：和好友一对一比拼
- 活动榜：每个活动内的排名
