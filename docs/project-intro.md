# HabitLoop — 让习惯养成像游戏一样有趣

> 用游戏化的方式，让打卡不再枯燥，找到志同道合的伙伴一起成长

---

## 产品简介

HabitLoop 是一款游戏化的习惯养成应用。我们相信，最好的动力来自**有趣的体验**和**伙伴的支持**。

通过积分、成就、徽章和社群系统，让养成习惯变得有趣，而不是负担。

## 三大预设活动

### 🏋️ 健身/运动打卡

**目标类型**
- 🏃 每日跑步（5km/10km/半马）
- 🧘 每日瑜伽（10分钟/30分钟）
- 💪 健身房打卡（胸部/背部/腿部训练日）
- 📱 每日步数（5000步/8000步/10000步）

**每日任务举例**
- [ ] 运动时长达到30分钟（来自Apple Health自动同步）
- [ ] 拍照记录今日状态（可选）
- [ ] 打卡成功 +10积分，连续打卡翻倍

**自动追踪**
- Apple Health 无缝同步步数、运动消耗、运动时长
- 支持连接 Garmin、Keep 等第三方数据

**社群活动**
| 活动名称 | 周期 | 参与人数 | 奖励 |
|---------|-------|---------|------|
| 30天月跑挑战 | 每月 | 不限 | 完成者专属徽章 + 100积分 |
| 21天减脂营 | 每月 | 50人 | 减重最多的3人获大奖 |
| 每日早起打卡小队 | 长期 | 不限 | 连续7天早起解锁早起达人称号 |

## 核心功能

### 1️⃣ 智能目标设定
输入你的大目标，HabitLoop 自动帮你拆解成每日可执行的任务
```
目标：3个月减重5公斤
↓ 自动拆解
每日任务：运动30分钟 + 记录饮食 + 睡前打卡
```

### 2️⃣ 连续打卡奖励
| 连续天数 | 奖励 |
|---------|------|
| 3天 | 银色徽章 |
| 7天 | 金色徽章 + 双倍积分 |
| 30天 | 钻石徽章 + 解锁专属皮肤 |
| 100天 | 传奇徽章 + 终身VIP特权 |

### 3️⃣ 社群活动

**任何用户都可以发起打卡活动**

发起一个活动，你只需要：
1. 填写活动名称和描述
2. 设置参与规则和奖励
3. 发布分享

**热门活动示例**

| 活动名称 | 发起人 | 参与人数 | 状态 |
|---------|-------|---------|------|
| 21天早起挑战 | @早起侠 | 328人 | 进行中 |
| 30天读书马拉松 | @书虫小队 | 156人 | 报名中 |
| 每日健身打卡组 | @肌肉铁军 | 523人 | 长期 |
| AI 学习入门营 | @AI探索者 | 89人 | 报名中 |

**活动规则示例**
```
活动：21天早起挑战
发起时间：2025-04-01
参与人数：无限制
打卡时间：6:00 - 9:00
奖励机制：
  • 连续7天：早起达人徽章 + 50积分
  • 连续21天：早起王者徽章 + 200积分
  • 全勤（21天不间断）：专属头像框

淘汰规则：
  • 中断1次：不淘汰，但连续天数清零
  • 中断3次：自动退出活动
```

找到你的同频伙伴，一起坚持！
