# HabitLoop — 让习惯养成像游戏一样有趣

> 用游戏化的方式，让打卡不再枯燥，找到志同道合的伙伴一起成长

---

## 产品简介

HabitLoop 是一款游戏化的习惯养成应用。我们相信，最好的动力来自**有趣的体验**和**伙伴的支持**。

通过积分、成就、徽章和社群系统，让养成习惯变得有趣，而不是负担。

---

## 为什么是 HabitLoop？

### 你的问题
- 新年立的Flag，两周就倒了
- 一个人坚持，真的好难
- 今天打卡了吗？总是忘记
- 没人监督，容易三天打鱼两天晒网

### HabitLoop 的解决方案
| 你的痛点 | HabitLoop 怎么解决 |
|---------|-------------------|
| 孤独难坚持 | 找到志同道合的伙伴，互相打卡、互相监督 |
| 容易忘记 | 智能提醒 + 连续打卡奖励机制 |
| 缺乏动力 | 游戏化等级、积分、成就徽章 |
| 不知道怎么做 | 目标自动拆解为每日任务 |
| 看不到进步 | 可视化成长曲线，见证每一天的改变 |

## 三大预设活动

### 🏋️ 健身/运动打卡

**目标类型**
- 🏃 每日跑步（5km/10km/半马）
- 🧘 每日瑜伽（10分钟/30分钟）
- 💪 健身房打卡（胸部/背部/腿部训练日）
- 📱 每日步数（5000步/8000步/10000步）

**每日任务举例**
- [ ] 运动时长达到30分钟（来自Apple Health自动同步）
- [ ] 拍照记录今日状态（可选）
- [ ] 打卡成功 +10积分，连续打卡翻倍

**自动追踪**
- Apple Health 无缝同步步数、运动消耗、运动时长
- 支持连接 Garmin、Keep 等第三方数据

- 防断卡道具：会员专属，可使用1次防断卡
