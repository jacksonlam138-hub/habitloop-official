# HabitLoop — 让习惯养成像游戏一样有趣

> 用游戏化的方式，让打卡不再枯燥，找到志同道合的伙伴一起成长

---

## 产品简介

HabitLoop 是一款游戏化的习惯养成应用。我们相信，最好的动力来自**有趣的体验**和**伙伴的支持**。

通过积分、成就、徽章和社群系统，让养成习惯变得有趣，而不是负担。

---

## 为什么是 HabitLoop？

### 你的问题
- 新年立的Flag，两周就倒了
- 一个人坚持，真的好难
- 今天打卡了吗？总是忘记
- 没人监督，容易三天打鱼两天晒网

### HabitLoop 的解决方案
| 你的痛点 | HabitLoop 怎么解决 |
|---------|-------------------|
| 孤独难坚持 | 找到志同道合的伙伴，互相打卡、互相监督 |
| 容易忘记 | 智能提醒 + 连续打卡奖励机制 |
| 缺乏动力 | 游戏化等级、积分、成就徽章 |
| 不知道怎么做 | 目标自动拆解为每日任务 |
| 看不到进步 | 可视化成长曲线，见证每一天的改变 |

## 三大预设活动

### 🏋️ 健身/运动打卡

**目标类型**
- 🏃 每日跑步（5km/10km/半马）
- 🧘 每日瑜伽（10分钟/30分钟）
- 💪 健身房打卡（胸部/背部/腿部训练日）
- 📱 每日步数（5000步/8000步/10000步）

**每日任务举例**
- [ ] 运动时长达到30分钟（来自Apple Health自动同步）
- [ ] 拍照记录今日状态（可选）
- [ ] 打卡成功 +10积分，连续打卡翻倍

**自动追踪**
- Apple Health 无缝同步步数、运动消耗、运动时长
- 支持连接 Garmin、Keep 等第三方数据

**社群活动**
| 活动名称 | 周期 | 参与人数 | 奖励 |
|---------|-------|---------|------|
| 30天月跑挑战 | 每月 | 不限 | 完成者专属徽章 + 100积分 |
| 21天减脂营 | 每月 | 50人 | 减重最多的3人获大奖 |
| 每日早起打卡小队 | 长期 | 不限 | 连续7天早起解锁早起达人称号 |

### 2️⃣ 连续打卡奖励

**每天打卡，奖励翻倍**

```
┌─────────────────────────────────────────────────────────────┐
│  连续打卡奖励机制                                         │
└─────────────────────────────────────────────────────────────┘

第1天    第2天    第3天    第4天    第5天    第6天    第7天
  +10      +10      +10      +10      +10      +10      +20  ←双倍
  ✓        ✓        ✓        ✓        ✓        ✓        ✓
                         └─────────────────────────┘
                              金色徽章解锁
```

| 连续天数 | 基础积分 | 奖励 | 累计积分 |
|---------|---------|------|---------|
| 第1-2天 | +10 | - | 20 |
| 第3天 | +10 | 银色徽章 | 30 |
| 第4-6天 | +10 | - | 60 |
| 第7天 | +20 | **金色徽章** | 80 |
| 第8-13天 | +10 | - | 140 |
| 第14天 | +20 | **白金徽章** | 160 |
| ... | ... | ... | ... |
| 第30天 | +50 | **钻石徽章 + 解锁皮肤** | 500 |
| 第100天 | +100 | **传奇徽章 + 终身VIP** | 3000 |

**连续中断怎么办？**
- 中断1次：连续天数清零，但累计积分保留
- 中断恢复：重新开始计算连续天数
- 防断卡道具：会员专属，可使用1次防断卡

### 3️⃣ 社群活动

**任何用户都可以发起打卡活动**

发起一个活动，你只需要：
1. 填写活动名称和描述
2. 设置参与规则和奖励
3. 发布分享

**活动类型**

| 类型 | 说明 | 示例 |
|------|------|------|
| **限时挑战** | 有开始和结束时间，完成即结束 | 21天早起、30天读书 |
| **长期坚持** | 无限期，可以随时加入/退出 | 每日健身、每日背单词 |
| **目标型** | 达成目标即结束 | 跑完100km、读完10本书 |
| **对抗型** | 成员之间PK竞争 | 周排行榜、月度冠军 |
