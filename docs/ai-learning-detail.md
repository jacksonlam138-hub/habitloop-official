## 第一阶段：AI 文字生成（1-2周）

### Week 1: ChatGPT 基础

**Day 1-3: 账号与界面**

| 任务 | 具体操作 | 打卡 |
|------|---------|------|
| 注册账号 | 访问 chatgpt.com 注册 | [ ] |
| 了解界面 | 熟悉输入框、历史记录、设置 | [ ] |
| 第一次对话 | 问10个不同问题，观察回答 | [ ] |
| 尝试高级功能 | 试试文件上传、语音输入 | [ ] |

**推荐提问（练习用）**
- "帮我写一封请假邮件"
- "总结这篇文章的核心观点"
- "帮我写一段 Python 代码"
- "用简单的英语解释什么是 AI"

**Day 4-7: Prompt 入门**

学习 Prompt 基础结构：
```
角色 + 任务 + 要求 + 上下文
```

**示例对比**
```
❌ 差：写个产品介绍

✅ 好：
你是一位专业的产品经理（角色）
请帮我写一款智能手表的产品介绍（任务）
要求：200字左右，突出健康监测功能（要求）
目标用户：25-35岁都市白领（上下文）
```

**每日练习**
- [ ] 用不同 Prompt 生成同一主题，对比效果
- [ ] 记录你觉得最好的 Prompt 模板
- [ ] 尝试让 ChatGPT 帮你优化 Prompt

### Week 2: Claude & 其他工具

**Claude vs ChatGPT 对比**

| 特点 | ChatGPT | Claude |
|------|---------|--------|
| 长文本处理 | 限 GPT-4 | 更强，支持200k+ |
| 代码分析 | 强 | 更强，适合代码审查 |
| 中文表达 | 好 | 更自然 |
| 价格 | 免费/$20 | 免费/$20 |

**国内工具推荐**

| 工具 | 特点 | 价格 |
|------|------|------|
| Kimi | 长中文优秀，20万字上下文 | 免费 |
| 通义千问 | 阿里出品，中文流畅 | 免费/付费 |
| 文心一言 | 百度出品，中文理解好 | 免费 |
| 智谱清言 | 清华出品，学术强 | 免费 |

**实战任务**
- [ ] 用 Claude 阅读一篇长文章并总结
- [ ] 用 Kimi 分析一份文档
- [ ] 对比不同工具对同一问题的回答

## 第二阶段：AI 绘图（2-3周）

### Week 1: Midjourney 入门

**准备工作**
- [ ] 注册 Discord 账号
- [ ] 加入 Midjourney 服务器
- [ ] 订阅（$10/月起）

**基础指令**
```
/imagine [prompt]
```

**第一次生成**
```
/imagine prompt: a cute cat sitting on a windowsill,
sunny day, realistic style
```

**每日练习**
| 天数 | 练习主题 | 打卡 |
|------|---------|------|
| Day 1 | 简单单物描述 | [ ] |
| Day 2 | 添加场景 | [ ] |
| Day 3 | 尝试不同风格 | [ ] |
| Day 4 | 调整 --ar 宽高比 | [ ] |
| Day 5 | 调整 --quality 质量 | [ ] |
| Day 6-7 | 自由创作 | [ ] |

### Week 2: Prompt 进阶

**Prompt 结构**
```
[主体] + [风格] + [细节] + [参数]
```

**示例**
```
cute cat, watercolor painting, soft pastel colors,
whimsical style, highly detailed --ar 1:1 --q 2
```

**风格参考**
| 风格 | Prompt 关键词 |
|------|--------------|
| 写实 | photorealistic, 8k, detailed |
| 动漫 | anime style, studio ghibli |
| 水彩 | watercolor painting, soft colors |
| 油画 | oil painting, classical |
| 像素 | pixel art, 8-bit, retro |

### Week 3: Stable Diffusion / 国内工具

**Stable Diffusion**
- 本地部署：需要显卡（推荐 RTX 3060+）
- 云端方案：Google Colab、LiblibAI

**国内 AI 绘图工具**

| 文心一格 | 百度出品，中文友好 | 免费 |
| 通义万相 | 阿里出品，风格多 | 免费 |

## 第三阶段：AI 工作流（2-3周）

### Week 1: 组合使用工具

**工作流 1: ChatGPT + Midjourney**
```
1. 用 ChatGPT 生成详细描述
   "描述一个未来城市的场景"

2. 用生成的描述作为 MJ Prompt
   /imagine [粘贴描述]

3. 反复迭代优化
```

**工作流 2: Claude + Notion**
```
1. 用 Claude 生成内容大纲
2. 复制到 Notion
3. 用 Notion AI 扩展每部分
4. 整理成知识库
```

### Week 2-3: 自动化工作流

**Make/Zapier 自动化**
```
触发：收到邮件
 ↓
AI：Claude 总结邮件内容
 ↓
输出：保存到 Notion
```

**AI + Excel**
```
1. ChatGPT 生成 Excel 公式
2. 解释公式的含义
3. 优化数据处理流程
```

**实战项目清单**

| 项目 | 工具 | 产出 | 预计时间 |
|------|------|------|---------|
| 生成一周社交媒体内容 | ChatGPT + MJ | 7条图文 | 2小时 |
| 制作演示文稿 | Gamma | 完整PPT | 1小时 |
| 搭建个人知识库 | Claude + Notion | 知识库框架 | 3小时 |
| AI 绘画日签 | MJ | 30张图 | 持续 |

## 常见问题

**Q: 没有付费工具怎么办？**
A: 每个类别都有免费替代品，先用免费版入门。

**Q: 学习时间不够怎么办？**
A: 每天只需15-30分钟，重点是持续。

**Q: Prompt 总是写不好怎么办？**
A: 多看别人的 Prompt，参加 Prompt 优化局。

**Q: 什么时候算学会了？**
A: 能独立用 AI 完成一个完整项目即可。
