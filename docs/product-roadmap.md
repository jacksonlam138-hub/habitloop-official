# HabitLoop 产品路线图

## 概述

本文档描述 HabitLoop 从 MVP 到完整产品的开发路线规划。

---

## 阶段一：MVP 验证（当前）

**目标**: 收集用户付费意愿，验证市场需求

| 状态 | 任务 |
|------|------|
| ✅ 完成 | 项目介绍文档 |
| ✅ 完成 | Landing Page 开发 |
| ✅ 完成 | 预注册表单（Notion 集成） |
| ✅ 完成 | 部署指南 |
| 🔄 进行 | 上线收集数据 |
| ⏳ 待办 | 数据分析与决策 |

**决策点**: 收集到 100+ 有效预注册后，决定是否启动完整开发

---

## 阶段二：iOS APP 核心版（3-4个月）

**目标**: 上线 iOS 应用，实现核心打卡功能

### 功能清单

#### P0 - 核心功能
| 功能 | 说明 |
|------|------|
| 用户注册/登录 | Apple ID、邮箱注册 |
| 目标设定 | 创建目标、设定每日任务 |
| 每日打卡 | 打卡记录、上传图片 |
| 进度展示 | 连续天数、完成率 |
| 推送通知 | 打卡提醒 |

#### P1 - 游戏化基础
| 功能 | 说明 |
|------|------|
| 积分系统 | 打卡获得积分 |
| 等级系统 | 基于积分的等级 |
| 连续打卡奖励 | 3/7/30/100天徽章 |
| 成就系统 | 基础成就解锁 |

#### P2 - 三大预设活动
| 功能 | 说明 |
|------|------|
| 健身/运动 | Apple Health 同步 |
| AI 学习 | 预设学习路径 |
| 英语学习 | 单词/听力/口语任务 |

**上线目标**: App Store 审核 + 500+ 下载

---

## 阶段三：社群功能（2-3个月）

**目标**: 实现用户社交和活动组织功能

### 功能清单

#### 活动系统
| 功能 | 说明 |
|------|------|
| 创建活动 | 任意用户可发起活动 |
| 活动模板 | 提供常用模板 |
| 加入/退出活动 | 用户自由选择 |
| 活动管理 | 管理员权限 |

#### 社交功能
| 功能 | 说明 |
|------|------|
| 用户主页 | 展示个人信息和成就 |
| 添加好友 | 邀请码、搜索添加 |
| 互动点赞 | 给打卡内容点赞 |
| 评论功能 | 活动内讨论 |

#### 排行榜
| 功能 | 说明 |
|------|------|
| 活动排行榜 | 活动内排名 |
| 好友排行 | 好友间对比 |
| 全球排行 | 全平台排名 |

---

## 阶段四：游戏化深化（1-2个月）

**目标**: 增强游戏化体验，提升用户留存

### 功能清单

| 功能 | 说明 |
|------|------|
| 复杂成就系统 | 条件成就、稀有成就 |
| 主题皮肤 | 可兑换的 UI 主题 |
| 抽卡系统 | 盲盒式奖励机制 |
| 每日任务 | 额外任务奖励 |
| 挑战活动 | 限时活动 |

---

## 阶段五：智能化（3-4个月）

**目标**: 使用 AI 提升用户体验

### 功能清单

| 功能 | 说明 |
|------|------|
| 个性化推荐 | 推荐活动/伙伴 |
| 目标拆解 AI | 自动生成执行路径 |
| 智能提醒 | 基于行为的最佳提醒时间 |
| 学习路径生成 | 根据水平推荐学习内容 |
| 流失预测 | 识别流失风险用户 |

---

## 阶段六：Android 版本（2-3个月）

**目标**: 扩展到 Android 平台

### 方案选择

| 方案 | 说明 |
|------|------|
| React Native | 复用 iOS 代码，快速开发 |
| Flutter | 独立开发，性能更优 |
| 原生 Android | 最佳体验，需单独开发 |

**推荐**: 阶段二若采用跨平台技术，此阶段可缩短至 1 个月

---

## 阶段七：商业化（2-3个月）

**目标**: 实现可持续的商业模式

### 会员体系

| 等级 | 价格 | 权益 |
|------|------|------|
| 免费版 | ¥0 | 基础打卡、1个活动、基础数据 |
| 会员版 | ¥18/月 | 无限活动、数据分析、个性化推荐 |
| 高级版 | ¥68/月 | 1对1教练、专属活动、优先客服 |

### 增值服务
- 付费活动（如精品课程）
- 实体徽章/奖牌
- 企业团队版

---

### 阶段二：iOS 核心版

- **人力**: 1-2 人（开发 + 设计）
- **成本**:

  - 服务器/云服务: ¥500-1,000/月

### 阶段三-七
根据项目进展动态调整

## 风险与应对

| 风险 | 应对措施 |
|------|---------|
| 预注册数据不足 | 扩大推广渠道、调整定价策略 |
| App Store 审核不通过 | 提前了解规范、准备申诉材料 |
| 用户留存率低 | 强化游戏化、优化产品体验 |
| 竞品出现 | 强化差异化、快速迭代 |
